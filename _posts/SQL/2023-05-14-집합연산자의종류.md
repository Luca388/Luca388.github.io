# UNION
- 합집합    
> 중복을 제거한 결과의 합을 검색

# UNION ALL
- 합집합    
> 중복을 포함한 결과의 합을 검색

# INTERSECT
- 교집합

> 양쪽 모두에서 포함된 행을 검색

# EXCEPT(MINUS)
- 차집합 
> 첫 번쨰 검색 결과에서 두 번째 검색 결과를 제외한 나머지를 검색
